英语语法

语法概述

动词分类

  1. 实义动词
    (1). 按动作承受者划分：及物动词和不及物动词
    (2). 按状态动作划分：动作动词和状态动词
  2. (连)系动词：可看作状态动词或者不及物动词
  3. 助动词：不能独立作谓语动词
  4. 情态动词：不能独立作谓语动词
  5. 非谓语动词：不定式 、过去分词、现在分词、动名词
    其中，实义动词和(连)系动词可作谓语动词、助动词和情态动词可以辅助构成谓语动词、非谓语动词不能作为谓语动词；
    若按需要宾语补语（完全实义动词）的动词：
    (1). 认定动词；
    (2). 使役动词；
    (3). 任命动词；
    (4). ····

英语时态

时态分为动作时间和动作状态；
时间包括四种现在、过去、将来、过去将来；
状态包括四种一般、完成、进行、完成进行；
时间和状态可以随意组合为16种(理论)；

  1. 一般现在时态
    用法：
    (1). + 动词原型
    (2). + 动词原型 + s(第三人称单数)
    使用条件：
    (1). 表达事实；
    (2). 表达习惯/重复的动作；
    (3). 表示预计发生的事；
  2. 现在进行时
    用法：助动词be的变位 + 动词的现在分词
    现在分词：
    (1). 通常为v + ing
    (2). 或者v的不规则变化
    be：
    I am
    You/we/they are
    She/he/it is
  3. 现在完成时态
    用法：助动词have的变位 + 动词的过去分词
    have：
    (1). 实义动词；
    (2). 助动词，构成时态；
    I/you/we/they have
    She/he/it has
    过去分词：
    (1). 通常为v + ed
    (2). 或者v的不规则变化
  4. 现在完成进行时
    用法：have的变位 + been + 动词的现在分词
  5. 一般过去时态
    用法：+ 动词过去式
    过去式：
    (1). 通常为v + ed
    (2). 或者v的不规则变化

  6. 过去进行时态
    用法：助动词be的变位 + 动词的现在分词
    be：
    I/she/he/it was
    You/we/they were
  7. 过去完成时态
    用法：助动词have的变位 + 动词的过去分词
    have：I/you/we/they/she/he/it had
  8. 过去完成进行时态
    用法：助动词have的变位 + been + 动词的现在分词
    have：I/you/we/they/she/he/it had
  9. 一般将来时
    用法：will + 动词原型
  10. 将来进行时
    用法：will + be + 动词的现在分词
  11. 将来完成时
    用法：will + have + 动词的过去分词
  12. 将来完成进行时
    用法：will + have + been + 动词的现在分词
  13. 一般过去将来时态
    用法：would + 动词原型
    英语表达将来：助动词be的变位[was/were] + going + to + 动词原形也可表示过去将来时态
  14. 过去将来进行时态
    用法：would + be + 动词的现在分词
  15. 过去将来完成时
    用法：would + have + 动词的过去分词
  16. 过去将来完成进行时态
    用法：would + have + been + 动词的现在分词

动词语气（虚拟语气）

主要包含：“陈述语气、祈使语句，虚拟语气”

  1. 陈述语气
    主要描述现时
  2. 祈使语气
    主要描述命令/请求
    祈使语气一般把命令/请求用第二人称（“你”）用一般现在时态表达，然后去掉“你”，并保证动词原型
  3. 虚拟语气
    (1). 表示和现在事实相反；
    (2). 用来表示像这样的，将来时间里，可能性很小的动作；
    (3). 与过去已经发生的事实相反；
    两大用法：
    (1). 表不可能假设；
    a. 与现在相反
    条件句：过去式；
    主句：would + 动词原型；
    b. 与将来相反
    条件句：should[可使用were to 替代] + 动词原型；
    主句：would[should/could/might] + 动词原型；

    c. 与过去相反
    条件句：had + 动词过去分词；
    主句：would have + 动词过去分词；
    (2). 虚拟语气表达愿望，请求，建议，命令等=希望某个动作/事件发生；
    a. wish/hope：通常虚拟语气和表达愿望的词连用，wish/hope都表示愿望，wish表示可能性小的事情，hope比较中性
    wish：可以做对现在的愿望[wish + 动词过去式]；表达对过去的愿望[wish + had/would have + 动词过去分词]；表达对将来的愿望[wish + should/would/could/might + 动词原型]；
    b. would rather
    c. demand/suggest/insist/order···：这些词后面接的虚拟语气一律是should + 动词原型

非谓语动词

  1. 动词不定式
    一般形式为to + 动词原形；
    (1). 不定式作主语
    It is + 形容词 + for sb. + to do sth.
    it作形式主语；
    (2). 不定式作宾语
    使用see/find/watch/make/let/···后面用不定式作宾补需要省略to；
    (3). 不定式作定语
    (4). 不定式作状语
    a. 表原因
    b. 表目的
    c. 表结果
  2. 动名词
    一般形式为动词原型 + ing
    (1). 动名词作主语
    (2). 动名词作主语补语
    (3). 动名词作定语
    (4). 动名词作宾语
  3. 现在分词
    一般为动词原型 + ing
  4. 过去分词
    一般形式为动词原型 + ed；

独立主格

  1. 独立主格的形式
    (1). 名词 + 现在分词
    (2). 名词 + 过去分词
    (3). 名词 + 动词不定式
    (4). 名词 + 名词
    (5). 名词 + 形容词
    (6). 名词 + 副词
    (7). 名词 + 介词短语
    (8). ···

助动词&情态动词

情态动词起辅助主动词的作用；不能单独作谓语动词；

助动词的分类：

  1. 基本助动词
    (1). be：存在
    用法：be变位辅助构成XX时态
    (2). do：做
    用法：do变位辅助构成XX时态
    (3). have：有
    用法：have变位辅助构成XX时态
    这三个除了作助动词之外，本身还可以作实义动词，也就是可以单独作谓语动词；
  2. 情态助动词
    (1). can/could：有能力做····、会···；可以···；
    can/could做“许可”意义时，虽然could是can的过去式，但是没有实际时间意义，说法更礼貌委婉；
    can/could表可能性时，could表示事物可能性更低；
    can时态只有现在与过去时，一般表示现在和过去的能力；
    (2). may/might
    may/might表请求和许可时，might更礼貌；
    may/might表可能性时，表示可能性的程度小于can/could，也许、大概；
    may/might祝愿时，通常放句首,一般只会使用may，不使用might；
    (3). will/would
    will/would可以构成将来时；
    will/would可以表示请求，建议；
    will/would可以表示推测，假设；
    (4). shall/should
    shall可以构成将来时；
    shall可以表示请求或征求意见；
    shall可以表示警告，命令；
    should可以表示应当；
    should可以表示有一定根据的推测；
    should可以表示“竟然”；
    should可以构成虚拟语气；
    (5). must
    可以表示必须；
    可以表示禁止：一般使用“must not(mustn't)”只有必须不的意义=禁止；
    可以表示推测：表推测时，must只能用在肯定句中，否定只能用can’t；
    (6). ought to
    表示应该；
    (7). dare/dared
    dare做实义动词时，表示敢；挑战；冒险；
    dare做情态动词时，表示敢，一般用于否定句和疑问句中；
    (8). need
    做实义动词时，need表示“需要”，做相反意义时，前使用do变位；
    做情态动词时，need表示“需要，有必要”，做相反意义时，可以直接在后面 + not；
    (9). used to
    表示过去的动作，强调与现在对比
    注意：will/would和shall/should在某些传统语法体系中既属于“助动词”又属于“情态动词”；
  3. 半助动词
    包含：be able to、 be going to、 be about to、 had better···
    (1). be able to
    可以表达将来的能力，若是was/were able to···往往表示“成功做成···”；
    (2). be going to, be about to, be to
    表示将要做某事；
    be going to 一般是“很快就要做某事”；
    be about to 一般表示“马上做某事”；
    be to 一般表示“被命令做某事”；
    (3). had better
    表示“最好做···”
  4. 总结

使役动词

翻译为“使···，让···，令···”
常见使役动词：make/let/have/get···

  1. make
    make后面动词原型/裸不定式
  2. let
    允许某人做某事（不绝对）
    (1). let后面动词原型/裸不定式；
    (2). let后面副词；
  3. have
    有/助动词/使役动词
    作使役动词一般为“给某人一个任务/让某人干某事”；
    (1). have后面动词原形/裸不定式；
    (2). have后面过去分词；
  4. get
    (1). get后面(完整)不定式；
    通常：“说服/鼓励某人做某事；”
    (2). get后面过去分词；

从句

  1. 形容词从句
    由关系词（关系代词、关系副词）引导从句
    关系代词which/that一般可以互换；当先行词是唯一的用that，先行词多选一使用which；
    (1). 限定性形容词从句：限定范围、性质
    (2). 非限定形容词从句：指代整句话
  2. 名词从句
    (1). 主语从句
    (2). 宾语从句
    确定信息/不确定信息都可以作宾语从句；
    引导词whether/where/when/how作引导词时不是疑问代词/疑问副词，之后不加助动词
    当引导词为that时，引导词可以省略；
    在宾语从句中，否定加在(主句)谓语动词前，而不是从句中；
    宾语从句主从句的时态要保持一致；
    be + 形容词 + that···
    (3). 表语从句
    用一个从句表达系动词后的表语；
    系动词一般为be动词；
    (4). 同位语从句
    经常修饰抽象的事情；
    (5). 宾语补语从句
    将宾语补语使用从句替换；
  3. 副词从句
    起副词作用，补充说明；同称状语从句；
    (1). 时间副词从句
    a. 在某一时间点
    引导词：before
    b. 在某一时间点之中（当时）
    引导词：when/while/as
    when：强调时间点，“突然”
    while：强调时间段
    as：强调同时
    c. 在某一时间点之后
    引导词：after
    d. 从之前某一时间点开始算
    引导词：since
    一般使用完成状态
    e. 直到之后某一时间点
    引导词：until
    f. 一···就···
    引导词：as soon as
    g. 下次···
    引导词：the next time
    (2). 地点副词从句
    引导词：where + “强调形式”[wherever:不管哪里/everywhere:任何地方/anywhere:所有地方]，表示地点
    wherever:强调“不管”哪里
    everywhere:强调“所有”地方，等效（定语从句）every place where
    anywhere:强调“任何”地方，等效（定语从句）at any place where
    (3). 比较副词从句
    a. 引导词：形容词比较级 + than，“更···”
    b. 引导词：as + 形容词 + as，“同样得···”
    c. the (比较级), the (比较级)，“越···越···”
    (4). 条件副词从句
    “主将从现”
    从现：从句使用一般现在时态
    从句比主句落后一个时间段，可以应用虚拟语气；
    引导词if(如果)/unless(除非)/provided(如果)/as long as(只要)/in case(如果)
    (5). 让步副词从句
    阐述某种情况/做出某种假设
    让步会进一步阐述“相反/相对”；“即使/就算···也···”
    no matter/regardless of + 疑问句
    (6). 方式副词从句
    从句表示与现实相反的假设
    (7). 原因副词从句
    引导词：because，介绍某个原因；
    引导词：since，介绍某个原因；
    引导词：as，表原因；
    表因果或语气时：because[引出实在的因果关系] > since[引出都知道的情况] > as[引出显而易见的事实]
    because引导的从句在主句前后都可以，since/as引导的从句只能放主句之前；
    for可以表示因果关系，但不构成原因副词从句，只能构成复合句；
    (8). 目的副词从句
    主句表示做了什么，从句表示要达到的目的；
    so that 不能置于句前；
    一般使用情态助词，更好体现目的性；

    (9). 结果副词从句
    引导词：so that；因此···结果···，解释说明主句的情况；
